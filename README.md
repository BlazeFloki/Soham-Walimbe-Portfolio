# Soham Walimbe – Portfolio

Personal portfolio website hosted on GitHub Pages. Built with vanilla HTML, CSS, and JavaScript.

## 🔗 Live Site

**[https://blazefloki.github.io/portfolio](https://blazefloki.github.io/portfolio)**

> Replace `portfolio` with whatever you name the repository on GitHub.

## 📁 Project Structure

```
Portfolio/

├── assets/
```

## ✨ Features

- Responsive design (mobile-friendly)
- Smooth scroll with active nav highlighting
- Scroll-reveal card animations
- Dark theme with GitHub-inspired palette
- Sections: Hero, About, Experience, Projects, Skills, Education, Contact
